Lesson 1 – Cascading Stylesheets Introduction

The purpose of CSS (Cascading Style Sheets) is explained in this course. It discusses the three layers of a webpage (content, presentation, and behavior), how CSS distinguishes between content and design, and the benefits of utilizing CSS, including time savings, simpler maintenance, increased design flexibility, and enhanced accessibility.

Lesson 2 – Structure and Types of CSS

The structure of CSS, including selectors, properties, and values, is the main topic of this lecture. It describes how CSS distinguishes between look and content and presents style sheets as sets of instructions for browsers. The syntax, applications, and benefits of the three CSS types—external, internal (embedded), and inline—are also covered.

Lesson 3 – Div and Span Tag

This lesson offers real-world examples of using internal, external, and inline styles to apply CSS. It describes the distinction between block-level and inline elements and presents the display property. Additionally, it covers how to apply certain styles and organize webpages using div and span tags as container elements.

Lesson 4 – CSS Rules, Properties, and Classes

This course demonstrates how to write CSS rules using declarations and selectors. It presents CSS classes and classifies style properties (fonts, colors, text, layout, lists, etc.). In order to style effectively, students learn how to define and apply classes as well as use element+class selectors and group selectors. It highlights accuracy and reusability in web design.

Lesson 5 – CSS Selectors

CSS selectors, which are used to identify and style particular HTML elements, are covered in Lesson 5. It presents six primary categories of selectors, each with a distinct use in web design: class, element (tag), ID, universal, group, and attribute selectors. The course gives examples and explains when to use each selector, such as utilizing ID for unique elements and class for many elements with shared styles.

Lesson 6 – CSS Behaviors

Lesson 6 teaches how to use CSS to control the size and visibility of elements on a webpage. It explains properties like width, height, and visibility to manage how content appears. It also introduces pseudo-classes that change the style of links when users hover, visit, or click them.
